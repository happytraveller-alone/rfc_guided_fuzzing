# RFC Document Information Extractor

## 项目概述

这个项目旨在从 RFC（Request for Comments）文档中抽取相关信息。目前，该工具主要功能是抽取 RFC 文档的标题。未来计划扩展功能，以提取更多信息，如 RFC 之间的相关关系等。

## 当前功能

- 抽取 RFC 文档标题

## 计划功能

- 抽取 RFC 文档之间的相关关系
- 提取 RFC 文档的关键词
- 生成 RFC 文档的摘要
- 建立 RFC 文档的引用网络
